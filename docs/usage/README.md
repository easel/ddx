## Common Commands

```bash
# Initialize DDx in a project
ddx init

# List available resources
ddx list

# Apply a template or pattern
ddx apply <resource>

# Diagnose project health
ddx doctor

# Update toolkit
ddx update

# Contribute improvements
ddx contribute
```
